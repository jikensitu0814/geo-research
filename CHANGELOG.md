# Changelog

All notable public research package changes are documented here.

## [0.1.0] - 2026-06-04

### Added

- Public AI Search / GEO research snapshot covering aggregated CSV tables, figures, methodology notes, and Rosetta evidence mapping.
- Public package validator for required files, private artifact checks, image references, Rosetta source references, `.gitignore` safety patterns, and internal-term leakage.
- GitHub issue templates for data errors, methodology discussion, benchmark proposals, and validator reports.
- Pull request checklist for validation, schema, Rosetta, figure-source, and data-safety review.
- Release checklist, citation metadata, maintainer notes, support guidance, and security reporting guidance.

### Not Included

- Raw AI responses.
- SQLite databases.
- Client-facing reports.
- Runtime traces or local execution logs.
- Environment files or credentials.
