# Data Limitations

## Excluded From Public Release

- SQLite databases.
- Raw model responses.
- Raw prompt/run exports.
- Full URL inventories.
- Environment variables and API keys.
- Internal run IDs, client IDs, and local execution metadata.
- Presentation render caches and local office profiles.

## Why These Are Excluded

The public package is designed to support research review at the aggregate level while reducing accidental disclosure risk. Brand names and cited domains are kept because they are part of the public web evidence layer, but raw traces and internal identifiers are not needed for public validation.

## Interpretation Caveats

This is a static research snapshot for 2026-04-27 to 2026-05-02. It should not be read as a live ranking dataset, commercial recommendation list, or universal conclusion for every sub-industry.
